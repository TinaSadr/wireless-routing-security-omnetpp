# Wireless Routing Security Simulation using OMNeT++

A computer networks simulation project focused on analyzing security vulnerabilities in wireless routing protocols using OMNeT++.

## Project Overview
This project investigates how security awareness can be incorporated into wireless routing protocol simulations. The study evaluates routing behavior under attack-like conditions such as fake route replies and malicious routing activity.

## Technologies Used
- OMNeT++
- Computer Networks
- MANET Routing
- AODV Protocol
- Network Simulation

## Concepts Applied
- Wireless routing protocols
- Routing security analysis
- MANET simulations
- AODV behavior analysis
- Packet routing
- Network vulnerability assessment

## Simulation Scenarios
- Static routing analysis
- Hop-count routing analysis
- AODV routing simulations
- Fake RREP attack simulations
- Route stability analysis
